# SOC Incident Management System

## Description
This project is a Java-based system that manages cybersecurity incidents using LinkedList, Queue, and Stack.

## Features
- Read data from text file
- Assign analysts to queues
- Process incidents in batches
- Display resolved analysts using stack
- GUI implemented using Java Swing

## How to Run
1. Compile all Java files
2. Run SOC_GUI_FULL.java
3. Ensure cyber_incidents.txt is in the same folder

## Data Structures Used
- LinkedList
- Queue (FIFO)
- Stack (LIFO)
